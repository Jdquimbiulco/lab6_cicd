# lab6_cicd

Repositorio creado a partir del proyecto local `lab6`. Contiene una API Express pequeña con rutas de prueba y correcciones aplicadas en `index.js`.

Rutas disponibles:

- GET / -> mensaje de estado
- GET /saludo/:nombre -> saluda al nombre pasado
- GET /suma?numero1=1&numero2=2 -> devuelve la suma

Para ejecutar:

```bash
npm install
npm start
```
